🚀 Project Overview

This is a Full Stack Inventory Management System built using:

Frontend: React (Create React App)
Backend: Node.js + Express
Database: MySQL
Deployment:
Frontend → Vercel
Backend → Render
Database → FreeSQLDatabase (MySQL)

The system allows users to manage inventory items, categorize them, and track stock availability with real-time updates.

🚀 Features
Add new inventory items
Select item types from dynamic dropdown (fetched from database)
Update existing items
Delete items
Track stock availability (Available / Out of Stock)
Purchase entry with multiple items support
Responsive UI
REST API integration between frontend and backend
